# **App Name**: Afrah Organic Mart

## Core Features:

- Product Catalog: Display a comprehensive catalog of organic products, including dates, gur, seeds, and spices.
- Product Details: Provide detailed product information, image galleries, and customer reviews.
- Shopping Cart: Enable users to add, remove, and manage products in a shopping cart.
- Checkout Process: Guide users through a secure and efficient checkout process, including location-based delivery charge calculation.
- Wishlist: Allows customers to add products to a wishlist for future purchase.
- Admin Panel: Allow the owner to manage the products and change the prices.

## Style Guidelines:

- Primary color: Forest Green (#388E3C) to evoke a sense of nature and organic freshness.
- Background color: Light Beige (#F5F5DC) to provide a clean and natural backdrop.
- Accent color: Earthy Brown (#A0522D) for highlights and calls to action.
- Body text: 'PT Sans' (sans-serif) for clear and accessible reading. Headline: 'Alegreya' (serif) for a more elegant feel.
- Use a set of minimalist icons that evoke a sense of natural, hand-crafted and organic goods.
- Employ a mobile-first, grid-based layout with a focus on clear visual hierarchy and user-friendly navigation.
- Implement subtle animations and transitions to enhance user engagement without being distracting.
